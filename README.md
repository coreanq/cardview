## 낱말 카드 앱 


# cardview  
 - 자라나는 우리 아이들을 가르치기 위한 낱말 카드 앱  

# 사용 환경 
 - android
 - iphone
 - (미확정) android TV?

## Development Environment
 - Windows 32bit
 - Qt 5.11.1
 - Qt Creator 4.7.0
 - V-play for Qt 5.11.1  

## Features
 - 동물/과일-야채  영어/한글 낱말 카드 지원
 - TTS 지원으로 영어/한글 텍스트를 음성 읽어주기 가능
 - 자동으로 문자 읽어 주기 지원
 - 영어/한글 음성 높낮이 설정 가능
 - 영어/한글 음성 타입 설정 가능
